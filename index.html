About this chatbot

This is a restaurant recommendation chatbot built using the Yelp Open Dataset (Philadelphia). It uses machine learning models for intent detection and a content-based recommender system to suggest restaurants based on your preferences.

Click the chat icon in the bottom-right corner to start a conversation!

Try asking:

  • "Recommend a sushi restaurant"
  • "Find me cheap pizza places"
  • "Find restaurants similar to Tuna Bar"
  • "Tell me about BAP"
  • "Find highly rated Korean restaurants"
  • "I want delivery"

Built with Dialogflow · Flask · Ngrok · Yelp Open Dataset
